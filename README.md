# S25 Transfer Sync

S25 Transfer Sync is a desktop application built with Tauri, React, and TypeScript. It provides a dual-pane file manager interface to seamlessly transfer (push and pull) files between your Mac and a Samsung Galaxy S25 Ultra (or other Android devices).

## Features
- **Dual-Pane Interface:** Easily browse local Mac folders and remote Android storage simultaneously.
- **Push & Pull Files:** Quickly transfer files from Mac to S25 Ultra or vice versa.
- **Finder-style Selection:** Support for multi-file selection using Cmd/Ctrl and Shift clicks.
- **Progress Tracking:** Real-time progress bar and status updates during batch file transfers.
